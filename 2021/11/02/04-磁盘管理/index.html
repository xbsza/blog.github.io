04_磁盘管理

发表于 2021-11-02 更新于 2021-11-03 分类于 应用服务 热度： ℃ 评论：
本文字数： 3.1k 阅读时长 ≈ 3 分钟

  电脑硬盘是计算机最主要的存储设备。硬盘（港台称之为硬碟，英文名：Hard Disk Drive， 简称HDD 全名温彻斯特式硬盘）由一个或者多个铝制或者玻璃制的碟片组成。这些碟片外覆盖有铁磁性材料。

环境安装

实验环境：rhel 7 64位
镜像：ISO镜像下载地址
实验需求：了解磁盘管理的原理以及创建分区 、格式化、挂载分区、开始使用的方法。

实验步骤

虚拟机内新添加一块硬盘

列出所有可用模块设备的信息

创建磁盘分区

刷新磁盘分区

[root@localhost ~]# partprobe /dev/sdb

再次查看分区列表

建立对应的文件目录以及分区格式

挂载新磁盘

[root@localhost ~]# mount /dev/sdb /tmp/disk/

查看是否已挂载

编辑配置文件 实现永久挂载
