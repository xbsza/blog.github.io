07_MySQL服务

  MySQL是一个关系型数据库管理系统 。它是当下最流行的关系型数据库管理系统之一，在 WEB 应用方面，也是最好的 RDBMS (Relational Database Management System，关系数据库管理系统) 应用软件之一。关系数据库将数据保存在不同的表中，而不是将所有数据放在一个大仓库内，这样就增加了速度并提高了灵活性。

环境安装

实验环境：Centos 7 64位
镜像：ISO镜像下载地址
实验需求：通过了解mysql原理以及搭建mysql服务远程登录连接mysql。

实验步骤

挂载镜像源

创建本地yum源仓库

下载并安装MySQL官方的 Yum Repository

[root@localhost mnt]# wget http://dev.mysql.com/get/mysql57-community-release-el7-10.noarch.rpm

[root@localhost mnt]# yum -y install mysql57-community-release-el7-10.noarch.rpm

[root@localhost mnt]# yum -y install mysql-community-server // 安装mysql客户端

启动Mysql服务

[root@localhost ~]# systemctl start mysqld.service

进入数据库之后 不能做任何操作 因为MySQL默认必须修改密码之后才能操作数据库 其中‘new password’替换成你要设置的密码
